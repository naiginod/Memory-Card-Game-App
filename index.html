  • Memory is a game of trying to match cards with the same number (A, 2, 3...10) or face (J, Q, K).
  • The user will first make a selection by clicking on one of the covered cards.
  • This selected card will stay exposed until you select the next card.
  • If the following selection is a match then both cards will disappear from the board.
  • If the following selection is NOT a match then both cards will return to being covered.
  • The user will continue this way until the board is cleared of all cards. Good Luck!
